# Quick Video Recording Guide

Fastest way to create your 2-minute project video.

## Quick Start (30 Minutes)

### Step 1: Install OBS Studio (5 minutes)
1. Download: https://obsproject.com/
2. Install and open
3. Add "Display Capture" source
4. Set resolution to 1920x1080
5. Click "Start Recording"

### Step 2: Prepare Demo (5 minutes)
1. Open your test interface: `test_kyc_bot.html`
2. Or prepare terminal with command ready
3. Have customer name ready: "Grigor Dimitrov"
4. Make sure service is running

### Step 3: Record (10 minutes)
1. Start OBS recording
2. Follow this simple flow:
   - [0:00-0:10] Say: "This is KYC Bot, an AI agent for KYC compliance"
   - [0:10-0:20] Show: Enter customer name, click "Run Investigation"
   - [0:20-1:00] Show: Investigation running, wait for results
   - [1:00-1:30] Show: Scroll through the report, highlight risk level
   - [1:30-1:50] Show: Open deployed service URL, test endpoint
   - [1:50-2:00] Say: "Check it out on GitHub. Thank you!"
3. Stop recording

### Step 4: Quick Edit (10 minutes)
1. Open Windows Video Editor (built-in)
2. Import your recording
3. Trim to 2 minutes
4. Add text overlays for key points
5. Export as MP4

## Even Simpler: Screen Recording Only

### Using Windows Game Bar:
1. Press `Win + G`
2. Click record button
3. Do your demo
4. Stop recording
5. Video saved to `Videos/Captures/`

### Using Loom:
1. Install Loom: https://www.loom.com/
2. Click "New Video"
3. Select "Screen + Camera" or "Screen Only"
4. Record your demo
5. Loom automatically saves and gives you a link

## Minimal Script (Just Do This)

1. **Say:** "This is KYC Bot, an AI agent that automates KYC compliance"
2. **Show:** Run an investigation with "Grigor Dimitrov"
3. **Show:** Wait for results (30-60 seconds)
4. **Show:** Scroll through the report
5. **Say:** "It uses three AI agents to search, check watchlists, and generate reports"
6. **Show:** The deployed service URL
7. **Say:** "Check it out on GitHub. Thank you!"

## What to Show

**Must Show:**
- ✅ The investigation running
- ✅ The final report
- ✅ The risk level
- ✅ The deployed service

**Nice to Show:**
- Architecture diagram
- Code snippets
- Search results
- Watchlist matches

## Tips

- **Don't worry about perfection** - judges want to see it working
- **Keep it simple** - focus on the demo
- **Show the results** - that's what matters
- **Be natural** - don't over-script it

## File to Upload

- **Format:** MP4
- **Length:** 1:50 - 2:10 (close to 2 minutes)
- **Resolution:** 1080p preferred, 720p minimum
- **Size:** Under 100MB if possible

## Done!

That's it! You don't need fancy editing - just show the agent working and explain what it does.
